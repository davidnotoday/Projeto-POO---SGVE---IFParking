----------------------------------------------------------------------------
O Projeto - Um sistema de gerenciamento das vagas de estacionamentos       |
----------------------------------------------------------------------------
E porque eu usaria um sistema de gerenciamento de vagas para meu estacionamento ?
Por um motivo simples,os usuarios teriam mais agilidade  de encontrar a sua vaga assim economizando bastante tempo.Com o método de seleção de vaga logo na entrada do estabelecimento, alem de o operador do sistema obter vários tipos de dados sobre o uso das vagas,como quais vagas são mais ocupadas,obter a média de tempo de uso de uma determinada vaga ou de setor,etc... o cliente obteria muito mais conforto.
----------------------------------------------------------------------------
Mais como funcionaria este sistema ? - O cliente ao chegar na cancela do  estabelecimento iria dispor de uma tela touch,aonde seria disponibilizada todas as vagas disponiveis e ocupadas,assim que o usuario selecionasse a sua vaga,será emitido um ticket com um código de barras ou o numero da sua vaga,quando o usuario fosse devolver o ticket o operador teria a informação do tempo que foi ultilizado a vaga e do valor que ele deverá receber depois disso seria dado baixa na vaga que estava ocupada e liberando para seleção para outro cliente na entrada do estabelecimento.
----------------------------------------------------------------------------
Será um sistema desenvolvido na linguagem de programação java ,ultilizando a programação orientada a objetos,juntamente com a linguagem android para a parde do cliente (tentativa , se não for possível será feita também em java)
